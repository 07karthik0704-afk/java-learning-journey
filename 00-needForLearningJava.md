# Why do we need programming?

Imagine you hire a new employee.

The employee is incredibly fast and never gets tired. But there's one problem...

**They do exactly what you tell them—nothing more, nothing less.**

A computer is like that.

It doesn't "think" the way humans do. It follows instructions.

Programming is simply **writing those instructions** in a language that humans can write and computers can eventually execute.

For example, when you click **Login** on a website, someone wrote a program that says:

```
If username exists
    and password is correct
        allow login
Else
        show "Invalid Credentials"
```

The computer isn't deciding. It's following instructions written by a programmer.

So a better definition is:

> **Programming is the process of writing instructions that tell a computer how to solve a problem or perform a task.**

---

# Why can't we just speak English?

Because computers don't understand English naturally.

Deep inside, a computer understands only **machine language**—patterns of 0s and 1s.

For example:

```
10110010
01101001
11001010
```

Imagine writing an entire banking application in binary.

Impossible for humans.

So we created programming languages that are easier for people to read and write.

# Why are there so many programming languages?

Think of vehicles.

* Bicycle
* Bike
* Car
* Bus
* Train

They all help people travel, but each is designed for different situations.

Programming languages are similar.

* **C** → Fast, close to hardware.
* **C++** → High-performance applications and games.
* **Python** → AI, automation, data science, scripting.
* **JavaScript** → Web browsers and interactive websites.
* **Java** → Enterprise software, backend systems, Android (historically), banking, large-scale applications.

Different tools for different jobs.

# Then why should I learn Java?

Because Java teaches you **how to think like a software engineer**, not just how to write code.

Java is popular because it is:

✅ Object-Oriented (OOP)

* Organizes code into objects and classes.
* Makes large projects easier to manage.

✅ Platform Independent

Java code is compiled into bytecode, which runs on the Java Virtual Machine. That's why people often say:

> **"Write Once, Run Anywhere."**

✅ Used in industry

Many companies use Java for:

* Banking systems
* E-commerce
* Enterprise applications
* Backend services
* Android apps (especially legacy projects)

Thousands of companies still rely on Java every day.
